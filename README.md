#### How Use :

##### =-> Download & Build :
```
cd ~/Desktop && git clone https://github.com/albinoz/ffmpeg-static-OSX.git && \
chmod 755 ~/Desktop/ffmpeg-static-OSX/ffmpeg-static-OSX.command  && \
~/Desktop/ffmpeg-static-OSX/ffmpeg-static-OSX.command
```
